Documentation

One page to orient a human or an agent quickly.

This surface is intentionally terse. It covers how to connect, what the runtime exposes, and what constraints govern every tool call.

01 · Connect

Use the live endpoint first; use local stdio for development.

Remote MCP endpoint:

https://arifosmcp.arif-fazil.com/mcp

Minimal MCP client config:

{
  "mcpServers": {
    "arifos": {
      "url": "https://arifosmcp.arif-fazil.com/mcp"
    }
  }
}
02 · Local development

The repo is the canonical source of truth.

git clone https://github.com/ariffazil/arifOS
cd arifOS
pip install -e ".[dev]"

# HTTP / SSE
python server.py

# STDIO for local MCP clients
python ops/runtime/stdio_server.py

`server.py` is the public entrypoint. `ops/runtime/stdio_server.py` is the minimal local stdio path used by assistant tooling.
